## tidyup

This tool is moving and categorizing your files into folders automatically and setting up your home directory

```
/home/a.rs
/home/b.c
/home/c.pdf
/home/d.doc
/home/Softwares
/home/Documents
/home/Music
```

```
$ tidyup --folder home
...
$ # Set up your folders easily!
```
